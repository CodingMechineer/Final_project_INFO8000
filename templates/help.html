Report Parameters

  • output - csv, json, or html.
    • If not provided, default to html, which should result in an ordinary HTML table of data.
    If a CSV is requested, it should generate a CSV file, and provide HTML with a link to the file.
    If json is requested, it should return JSON text.
  • start_date - only reports after this date
  • end_date - only reports before this date
  • lat, lng, dist - only reports within dist (km) of this lat/lng
  • max - only this number of reports, if not provided, return all of them
  • sort - if not provided, sort newest to oldest. If provided, sort as specified (oldest or newest)

Modify the URL to sort and choose the data regarding your needs:

Example URL: {{ url }}data?output=html&start_date=2022-01-01&end_date=2022-12-31&lat=40.7128&lng=-74.0060&dist=10&max=100&sort=newest
